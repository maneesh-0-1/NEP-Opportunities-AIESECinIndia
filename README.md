# NEP Opportunities Dashboard (AIESEC in India)
A premium, dark-themed React application built to elegantly showcase National Employer Partnership (NEP) opportunities for AIESEC in India.

### Tech Stack
- **Frontend Framework:** React + Vite
- **Styling:** Vanilla CSS (Glassmorphism UI)
- **Typography:** Bricolage Grotesque
- **Data Source:** Google Apps Script / Google Sheets API
